Chainplots


Graphical visualizations of blockchain data.

This website was built by Federico Laport Rivas as a Masters Dissertation applying for the Information Security Masters Degree at Universitat Autònoma de Barcelona. The dissertation was directed by Jordi Herrera Joancomartí. The source code can be reviewed in GitHub.

The project was written in Python. It is a Django application and some BlockSci analysis scripts.

Graphical visualizations.

Every parameter plotted is represented as a matrix of pixels and as a classic line graph. In the pixel matrix, each pixel represents a block of the bitcoin blockchain. This approach was heavily inspired by utxo-stats.com. In the other hand, we have a vertical shaped classic line graph. Up is present and down is past, following the notion of block depth.
